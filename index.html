About the Project

This site is designed to display 100 virtual coins and give you the ability to see live reports of their value.

On the home page, you'll see each of these coins. When you click on "more info," you can see the value of these coins in USD, NIS, and EUR. You can also select up to five coins and view live reports for them on a graph. Please note: If you try to select a sixth coin, you'll get a pop-up that will request that you unselect one of the coins in order to add a different one.

Once you've selected some coins, you can click on the "live reports" page in order to see a graph with each of the coins that you've selected. This graph will display the value of each of the coins every two seconds.
